Modelo para o Sensor CEI¶

Este dataset "DataCEI.csv" possui informaes dispostas em colunas sobre as caractersticas dos objetos que passam pelo sensor:

  • Tamanho: Segue a classificao do CEI2020 (Tamanho='0' - Grande 100%).
  • Referencia: Referncia dinmica do *Threshold.
  • NumAmostra: Nmero de amostras adquiridas.
  • Area: Somatrio das Amplitudes das amostras.
  • Delta: Mxima Amplitude da amostra.
  • Output1: Pea tipo 1.
  • Output2: Pea tipo 2.

Algoritmo Backpropagation¶

In [ ]:
epochs = 100000
last_loss=None
EvolucaoError=[]
IndiceError=[]

for e in range(epochs):
    delta_w_i_h = np.zeros(weights_input_hidden.shape)
    delta_w_h_o = np.zeros(weights_hidden_output.shape)
    for xi, yi in zip(X_train.values, y_train.values):
        
# Forward Pass
        #Camada oculta
        #Calcule a combinao linear de entradas e pesos sinpticos
        hidden_layer_input = np.dot(xi, weights_input_hidden)
        #Aplicado a funo de ativao
        hidden_layer_output = sigmoid(hidden_layer_input)
    
        #Camada de Sada
        #Calcule a combinao linear de entradas e pesos sinpticos
        output_layer_in = np.dot(hidden_layer_output, weights_hidden_output)

        #Aplicado a funo de ativao 
        output = sigmoid(output_layer_in)
        #print('As sadas da rede so',output)
#-------------------------------------------    
    
# Backward Pass
        ## TODO: Clculo do Erro
        error = yi - output
    
        # TODO: Calcule o termo de erro de sada (Gradiente da Camada de Sada)
        output_error_term = error * output * (1 - output)

        # TODO: Calcule a contribuio da camada oculta para o erro
        hidden_error = np.dot(weights_hidden_output,output_error_term)
    
        # TODO: Calcule o termo de erro da camada oculta (Gradiente da Camada Oculta)
        hidden_error_term = hidden_error * hidden_layer_output * (1 - hidden_layer_output)
    
        # TODO: Calcule a variao do peso da camada de sada
        delta_w_h_o += output_error_term*hidden_layer_output[:, None]

        # TODO: Calcule a variao do peso da camada oculta
        delta_w_i_h += hidden_error_term * xi[:, None]
        
    #Atualizao dos pesos na poca em questo
    weights_input_hidden += learnrate * delta_w_i_h / n_records
    weights_hidden_output += learnrate * delta_w_h_o / n_records
    
    
    # Imprimir o erro quadrtico mdio no conjunto de treinamento
    
    if  e % (epochs / 20) == 0:
        hidden_output = sigmoid(np.dot(xi, weights_input_hidden))
        out = sigmoid(np.dot(hidden_output,
                             weights_hidden_output))
        loss = np.mean((out - yi) ** 2)

        if last_loss and last_loss < loss:
            print("Erro quadrtico no treinamento: ", loss, " Ateno: O erro est aumentando")
        else:
            print("Erro quadrtico no treinamento: ", loss)
        last_loss = loss
         
        EvolucaoError.append(loss)
        IndiceError.append(e)
